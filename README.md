# skin_microbiome_16S_seq_analyses

This repository contains code for pre-processing and analyzing 16S rRNA sequencing data in a PhD project on skin microbiome of Norwegian soldiers. This work resulted in two papers: 1) *Skin bacterial community dynamics of hands and forearms before and after military field exercise*. Published in [Microbiology spectrum](https://journals.asm.org/doi/10.1128/spectrum.02953-24)
2) *Longitudinal analysis of the hand microbiome in response to chlorine-based antiseptic use during a military field exercise* Preprint in [bioRxiv](https://www.biorxiv.org/content/10.1101/2025.11.20.689467v1.full.pdf).
